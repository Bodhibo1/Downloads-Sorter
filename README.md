# Downloads Sorter

## It Sorts Your Downloads Folder Into Different Categories Based On File Types

By double clicking the .exe file, it will automatically find your downloads folder 
and sort most file types into different base categories, these being Images, Executables,
Gifs, Videos, Sounds, Zips, and Document Files.

Then it will create folders to house the new file locations. This is especially helpful
for decluttering and easily deleting old and useless setup .exe files and clearing space
on your computer.

For those who have python downloaded, by double clicking the bat file in the source folder 
it will do exactly the same thing, but you can easily modify the .py file and create new
categories or add new file types to the directory by adding to the dictionary labled "file_types"
